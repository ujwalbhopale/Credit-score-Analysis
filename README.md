Credit Risk Scoring System

Project Overview:
This project implements a full end-to-end Credit Risk Scoring System aligned exactly with the implemented notebook/code. 
It covers data ingestion from multiple sources, synthetic transaction enrichment, rigorous EDA, feature engineering, 
statistical feature selection (VIF, WOE/IV), multiple model families, imbalance handling, hyper-parameter optimization (RandomizedSearch & Optuna), and 
business-grade evaluation (ROC-AUC, KS, Gini), followed by Power BI consumption.
